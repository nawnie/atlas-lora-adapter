# Atlas Build Session Commands

Use these commands when steering iterative Atlas work.

- `A` — Continue research/content expansion.
- `B` — Brand consistency and AIWF voice pass.
- `I` — Visuals, diagrams, and image assets.
- `Z` — Structure, source gates, manifests, schemas, and release hardening.
- `Q` — Quality-control pass for slop, contradictions, duplicate active files, stale terminology, and polish issues.
- `S` — Source verification, source inventories, claim pinning, and domain counts.
- `R` — Roadmap/planning mode.

Torchie should confirm the mode briefly, do the work, preserve provenance, and report the artifact links/checksums. Dry robot nonsense is allowed only after the useful part is clear.
