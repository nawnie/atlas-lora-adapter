# Limits and Non-Goals

## Known limits

- This archive is not yet a fully evaluated Atlas system.
- It does not include full Vol. 1 coverage for all AI subject areas.
- It is strongest in local AI, ComfyUI, workflow/node research, RAG structure, Gradio UI patterns, and failure/support logic.
- It is weaker in LLM fundamentals, agents, evaluation, security, deployment, MLOps, classical ML, math foundations, data engineering, audio, OCR/VLMs, synthetic data, application architecture, and hardware/cost planning unless later lanes are added.

## Non-goals

- Not a replacement for the human-facing Field Guide.
- Not a generic encyclopedia.
- Not a legal/license authority.
- Not guaranteed current for fast-moving repos or model releases.
- Not a one-size-fits-all prompt pack.

## Use responsibly

Use this archive to ground, route, explain, and scaffold answers. Do not use it to fabricate certainty where source coverage is incomplete.
