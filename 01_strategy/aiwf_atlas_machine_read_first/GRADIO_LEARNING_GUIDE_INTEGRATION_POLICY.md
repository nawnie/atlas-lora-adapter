# Gradio Learning Guide Integration Policy

The Gradio learning guide and Topics 20–25 expanded materials are integrated as a source pack and cross-lane routing layer.

Use them for:

- learning path generation
- lab and capstone planning
- Gradio UI patterns
- cross-topic application design
- glossary/diagram retrieval
- machine routing between Topics 20–25

Do not use them as sole proof for current claims about model rankings, hardware metrics, release dates, benchmark percentages, or licensing. Route those claims through the web verification policy and prefer official documentation, primary papers, or vendor posts.

When answering with this source pack, prefer this order:

1. Relevant canonical lane overview and Atlas cards.
2. Gradio learning guide source pack.
3. Official/current source verification anchors.
4. Raw historical or provenance material only when the user asks for history.
