# Machine Governance Summary

This adapter is a machine-facing knowledge layer for AI Without Fear. It is not a casual reading package.

Core rules:
- Prefer canonical research lanes over raw source archives.
- Prefer newer, consolidated, cited, and lane-indexed material over legacy project drafts.
- Preserve citations and source trails whenever possible.
- Treat fast-moving model rankings, package status, and hardware claims as freshness-sensitive.
- Do not collapse project policy, user notes, and sourced facts into one confidence level.
- When knowledge conflicts, surface the conflict and route by source priority.
- Use failure-signature and Atlas-card files as routing aids, not as final answer substitutes.
