# Scope — LLM Fundamentals

## In scope
- tokenization and chat templates
- transformer architecture and attention
- context windows and memory limits
- sampling controls and logits
- instruction tuning and preference alignment
- quantization and local model tradeoffs
- small-model behavior under retrieval

## Out of scope / caution
- frontier model ranking as permanent truth
- vendor hype without reproducible behavior
- deep mathematical derivations beyond practitioner utility

## Vol. 1 target
This lane should give a small or mid-sized assistant enough domain scaffolding to answer practitioner-level questions, route to better source material, and identify when current web verification or expert review is required.
