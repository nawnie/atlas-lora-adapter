# Scope — MLOps, LLMOps, and Release Management

## In scope
- experiment tracking
- model registry
- prompt versioning
- dataset versioning
- evaluation versioning
- deployment stages
- rollback strategy
- CI/CD for AI apps
- release gates
- monitoring drift
- feedback loops
- change logs

## Out of scope / caution
- enterprise ceremony without practical value
- untracked prompt edits in production
- shipping without eval baselines

## Vol. 1 target
This lane should give a small or mid-sized assistant enough domain scaffolding to answer practitioner-level questions, route to better source material, and identify when current web verification or expert review is required.
