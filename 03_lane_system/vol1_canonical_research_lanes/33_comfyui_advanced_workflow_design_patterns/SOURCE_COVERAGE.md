# Source Coverage — ComfyUI Advanced Workflow Design Patterns

Primary coverage comes from official ComfyUI documentation for subgraphs, subgraph blueprints, workflow JSON, nodes, built-in switch nodes, lazy evaluation, node expansion, node replacement, and workflow templates.

The lane intentionally avoids relying on forum examples as primary evidence. Community examples may be useful for real-world workflow patterns, but they should not override official datatype, execution, subgraph, or migration behavior.
