# Scope — Agent Systems, Tool Use, and MCP

## In scope
- tool/function calling
- structured outputs
- MCP clients and servers
- resources, tools, and prompt surfaces
- planner-executor patterns
- state machines and LangGraph-style workflows
- agent memory boundaries
- tool permission and sandboxing

## Out of scope / caution
- autonomous real-world action without authorization
- unsafe tool execution
- treating agents as magic instead of stateful software

## Vol. 1 target
This lane should give a small or mid-sized assistant enough domain scaffolding to answer practitioner-level questions, route to better source material, and identify when current web verification or expert review is required.
