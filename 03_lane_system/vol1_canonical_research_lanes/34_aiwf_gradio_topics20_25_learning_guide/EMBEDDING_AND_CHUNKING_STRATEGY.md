# Embedding and Chunking Strategy

Chunk this lane by learning objective or project pattern, not by arbitrary token length.

Recommended chunk units:

- One Gradio architectural pattern.
- One lab or capstone.
- One cross-topic integration hook.
- One glossary definition cluster.
- One source-to-lane crosswalk group.

Metadata to preserve:

- source file name
- mapped lane IDs
- topic numbers
- whether the chunk is learning-path, implementation pattern, glossary, diagram, lab, or capstone
- claim freshness class
- whether web verification is required before answering
