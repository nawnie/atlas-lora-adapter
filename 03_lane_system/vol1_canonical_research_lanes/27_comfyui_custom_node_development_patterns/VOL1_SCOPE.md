# Vol. 1 Scope — Custom node development patterns

## Scope

This lane covers INPUT_TYPES, RETURN_TYPES, FUNCTION, CATEGORY, hidden inputs, lazy evaluation, V3 migration, packaging, and compatibility safety.

## Non-scope

This lane does not replace official ComfyUI documentation or runtime source inspection. Current ComfyUI behavior should be verified against official docs/source before release-sensitive claims are made.

## Machine-use guidance

Use this lane when the user asks practical advanced ComfyUI questions in this focus area. Prefer concrete examples, explicit socket/datatype names, and safety warnings when graph or node changes can break execution.
