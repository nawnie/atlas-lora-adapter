# Atlas Before the Adapter

Status: public-facing concept note  
Version added: v0.2.9-dev19  
Date: 2026-05-23

The Atlas project does not need the LoRA to be the first useful artifact.

A smaller test comes first:

```text
Can an agent use the Atlas protocol better than a chat attachment?
```

That can be tested with:

```text
AGENTS.md
repo instructions
source rules
report templates
optional Codex plugin skill
```

If that works better than dropping the Atlas into chat as an attachment, it suggests the map matters.

If a later Qwen LoRA fails, that does not automatically mean the map failed.

It may mean the training variables are wrong.

The LoRA experiment still matters.

It is just not the only way to test the Atlas idea.

No trained adapter exists yet.

No agent comparison has been run yet.

This is a smaller lane inside the whole concept.
